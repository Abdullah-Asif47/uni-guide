# 🎓 Pakistan University Guide

An AI-powered chatbot that helps Pakistani students navigate university admissions, merit lists, and career decisions.

## Features
- Ask questions about any Pakistani university in English or Urdu
- Get guidance on admissions, merit ranges, and programs
- Career path suggestions after FSc/A-Levels

## Tech Stack
- Python / Django
- Groq API (LLaMA 3.3 70B)
- Tailwind CSS

## Setup
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add your Groq API key to `.env`: `GROQ_API_KEY=your_key`
4. Run: `python manage.py runserver`
